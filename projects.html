Pattern Matching Program

This program checks if a given string contains a specified pattern as a substring. It's an example of basic Java string manipulation and algorithm implementation. 

        public static boolean s1ContainsS2(String s1, String s2) {
            boolean matchFound = false;
            int length1 = s1.length();
            int length2 = s2.length();
        
            if (length2 > length1) return matchFound;
        
            for (int i = 0; i <= length1 - length2; i++) {
                int j;
                for (j = 0; j < length2; j++) {
                    if (s1.charAt(i + j) != s2.charAt(j)) {
                        break;
                    }
                }
                if (j == length2) {
                    matchFound = true;
                    break;
                }
            }
            return matchFound;
        }
